The Northern Tunnels

The first game I took part in making. The northern tunnels taught me some fundamental rules and knowledge in game design needed to work with game development. Here's where I took my first steps into learning code. Working in gamemaker was certainly a learning experience.

The game

The Northern Tunnels is a top down roguelike game, featuring a main character with skills such as archery and sword/dagger fighting. In order to complete the game you would have to defeat all enemies and aquire a key to find your way to the outside world.

Gamemaker studio

To help with learning fast, and getting a grasp over working with game development, the game engine of choise was gamemaker.

Game design

While the group had a product in mind, a lot of this project ended up being a way for us individually to start thingking like game designers. The product didn't end up being all that playable, however I was able to learn and balance important game design concepts such as tools of play, victory and lose conditions, luck, and strategy.

Hugibert's Time Warp

My latest project, and undeniably the most serious project i've taken part in. I acted "chairman" and organized team meetups and scrum. Most of the time i was implementing assets into the game engine, and scripting animations, input and audio implementation.

The game

Hugibertâ€™s Time Warp is a top down roguelike game set in a large factory with rooms and floors to conquer and explore. The main character is a Scottish farmer [Hugibert Pain McFinnigan] from a medieval village [Tullich]. He is thrown into a modern day factory setting, making the theme a unique clash between two different eras and styles.

Unity

In making Hugibert's Time Warp I was fortunate enough to learn many elements of the game engine unity. Together with prior knowledge and simultaneous learning of C#, my main role in this project became coding a lot of parts for the game, such as player input, scene and audio-management, animation control.

Animations

Thoughtfully put together animator layers with blend trees was used for controlling each of the game's characters animations. Unity's ability to set script's variables thorughout the animations playback made me able to set up callback methods for actions such as attacks, movement etc.

Audio system

I also made the sound effects for the game, and took care of how the audio should be controlled through scripts.
